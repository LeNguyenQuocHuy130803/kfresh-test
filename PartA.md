## Part A – Multiple Choice

**1. Which of the following CSS properties is used to create a flexible box layout?**  
B. display: flex

---

**2. In JavaScript, what does the === operator check?**  
C. Both value and type

---

**3. What is the correct way to add a comment in HTML?**  
C. <!-- This is a comment -->

---

**4. In React, what hook is used to manage local component state?**  
C. useState

---

**5. Which unit in CSS is relative to the font-size of the root element?**  
C. rem

---

**6. What is the purpose of the “key” prop in React lists?**  
B. To help React identify which items have changed

---

**7. Which of the following is NOT a valid HTTP method?**  
C. FETCH

---

**8. In React Native, which component is equivalent to a `<div>` in web development?**  
C. View

---

**9. Which CSS property controls the stacking order of elements?**  
B. z-index

---

**10. What does the acronym “API” stand for?**  
A. Application Programming Interface
